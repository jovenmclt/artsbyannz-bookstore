# Artsbyannz Bookstore

A web-based bookstore and social platform where users can explore books, interact with the admin, and purchase books online. Admins can manage books, post updates like social media, and communicate with members directly.

## About
Artsbyannz Bookstore combines e-commerce with community interaction. Users can browse and purchase books, view admin posts, and chat with the admin. Admin can manage book listings, create posts, handle payments, and engage with users through messaging.

## Features
- **User Role**
  - Register and login
  - Explore books and admin posts like a social feed
  - Add to cart premade books
  - Purchase books online via Stripe and PayPal
  - Send and receive messages with Admin
- **Admin Role**
  - Manage book listings (add, edit, delete)
  - Post updates and announcements like social media
  - Handle online payments and transactions
  - Send and receive messages with members
  - Access full dashboard for oversight
  - Analytics and reports

## Technologies Used
- Frontend: Vue.js 3, Bootstrap 5, Inertia.js  
- Backend: Laravel 11  
- Database: MySQL  
- Payment: Stripe, PayPal  
- Other Tools: Composer, NPM
